			   INSTALLATION INSTRUCTIONS

After extracting AllJoyn C++ Code Generator to the desired location, run the
build.sh script.  This will create an executable named "ajgen" in the
specified directory.

    $sh build.sh [destination directory]

If no directory is specified, the executable  will be placed in the current
working directory.

The usage of the tool can be found by simply running the tool without
arguments:
$./ajgen
AllJoyn code-generation Tool
ajgen -h | --help
	Display this help
...
$
